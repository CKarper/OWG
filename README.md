# Open Work Graph

Open Work Graph (OWG) is an open, durable identity layer for intellectual works.

OWG exists to provide stable identifiers, reconciliation, provenance, and transparent stewardship for Works, Expressions, Manifestations, Agents, Subjects, and related entities.

The primary product of OWG is identity.

Everything else is built on top.
