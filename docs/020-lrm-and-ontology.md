# LRM and Ontology

## Principle

OWG does not create a new ontology for intellectual works.

OWG adopts IFLA LRM and builds on existing library science rather than replacing it.

## Working Model

LRM owns meaning.

OWG owns identity.

## Core Entities

- Work
- Expression
- Manifestation
- Item
- Agent
- Subject

OWG identifiers may be assigned to entities represented within the LRM model.

## Rationale

The hard problem is not defining what a Work is.

Libraries have spent decades refining those concepts.

The hard problem is creating durable, public, interoperable identity and reconciliation across systems.
