# Architectural Decision Records

Architectural Decision Records (ADRs) capture durable Open Work Graph design decisions and the rationale behind them.

ADRs are intended to preserve decisions that affect OWG identity, resolver independence, LRM alignment, reconciliation, governance, or namespace authority.
