# Non-Goals

OWG is not:

- A bookstore
- A publisher
- A DRM platform
- An ebook reader
- A file hosting service
- A replacement for Wikidata
- A replacement for Open Library
- A replacement for library catalogs
- A replacement for IFLA LRM
- A universal knowledge graph
- A social network

OWG focuses on durable identity, provenance, reconciliation, and stewardship for intellectual works and related entities.
